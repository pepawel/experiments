Cloudflare Subscription Checker

This tool checks if a website uses a paid Cloudflare subscription. How does it work?

The tool examines the /cdn-cgi/trace endpoint to detect Encrypted Client Hello (ECH) configuration.

Technical details: Cloudflare allows disabling ECH only in paid plans. When ECH is disabled (sni=plaintext), it indicates a paid subscription.
